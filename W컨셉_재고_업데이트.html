Work Rules

작업 규칙 · 12가지

파일 3개를 업로드하면 아래 규칙으로 자동 처리됩니다.

1
셀메이트 재고 CSV, SBR 선주문 파일, W컨셉 재고관리 파일 3개를 입력받아 KR재고수량을 자동 업데이트해줘.
2
가용재고 = 현재재고 − 미발송수 + SBR 선주문 잔여수량. SBR에 없으면 선주문수량 0으로 처리해줘.
3
스킵 셀메이트 품절 / 마진율 1% / 재고 동일 시 스킵. 단 셀메이트 미품절이면 W컨셉 품절이어도 재고 반영해줘.
4
M열 오른쪽에 검증 / 현재재고 / 미발송수 / 선주문수량 / 가용재고 5열 삽입. 변경 셀은 빨간색 Bold.
5
예약상품 + 실재고 > 0 → 바로배송으로 변경, 일반상품 + 실재고 < 0 → 예약배송으로 변경.
6
삽입한 5개 열의 1행 헤더를 연빨간색 배경으로 칠해줘. 확인 후 직접 삭제할 열이야.
7
1행에 필터를 걸어줘.
8
1행 고정 + 모든 열 너비를 가장 긴 글자 기준으로 자동 조절해줘.
9
전체 글자체 Calibri, 9pt로 통일해줘.
10
작업 후 [예약배송으로 변경할 상품], [바로배송으로 변경할 상품]을 상품명·옵션 한 줄씩 표시. 없으면 "없음.", 중복 상품명 제거.
11
재고관리_최종.xlsx(내부 검수용)와 함께 W컨셉_업데이트.xlsx(W컨셉 업로드용)도 생성. 변경 행만 담고 E, F, H, I열 비워둬.
12
상품명에 "마르셀 실키 반목 T" 포함 시 항상 스킵해줘.
